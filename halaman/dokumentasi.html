Dokumentasi

Struktur Folder

  • gaya/ — file CSS
  • halaman/ — konten setiap halaman
  • rancangan/ — komponen layout (header, footer, navbar, sidebar)
  • sistem/ — engine routing (app.js, navigasi.js, kelola.js, rakit.js, rute.json)

Menambah Halaman Baru

1. Buat file HTML baru di folder halaman/, misalnya kontak.html.

2. Daftarkan rutenya di sistem/rute.json:

"/kontak": {
  "judul": "Kontak",
  "header": "header.html",
  "main": "kontak.html",
  "css": "gaya.css"
}

3. Tambahkan link navigasi di header atau halaman manapun:

<a href="/kontak">Kontak</a>

Catatan Deploy

Karena ARKAJS menggunakan history.pushState, pastikan hosting yang digunakan mendukung fallback ke index.html untuk semua path (misalnya Cloudflare Pages, Netlify, atau Vercel).

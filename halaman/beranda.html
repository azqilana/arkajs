Starter kit client-side routing

Bikin project baru tanpa
build tool, cuma HTML, CSS, JS

ARKAJS adalah starter kit untuk project kecil yang sudah punya struktur folder, routing client-side, dan generator rute/script siap pakai — tanpa bundler, tanpa konfigurasi.

Baca dokumentasi
⚡

Langsung jalan

Satu command, project siap. Tidak perlu install bundler, webpack, atau framework besar.

🧭

Routing tanpa reload

Navigasi antar halaman pakai pushState, konten dimuat lewat fetch tanpa reload penuh.

🛠️

Generator bawaan

Bikin rute dan file logic baru lewat satu command, tidak perlu bikin file manual satu-satu.

🧩

Murni, tanpa dependensi

Cuma HTML, CSS, dan JavaScript bawaan browser. Tidak ada library tambahan yang perlu dipelajari.

👁️

runDOM siap pakai

Wrapper kecil yang aman dipakai untuk elemen yang baru muncul setelah navigasi, tanpa race condition.

📦

Siap di-deploy

Tinggal upload ke Cloudflare Pages atau hosting statis lain, sudah didukung dokumentasinya.

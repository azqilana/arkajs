Selamat Datang di ARKAJS

Mini framework client-side routing yang ringan, tanpa dependency, dan dibangun murni dengan HTML, CSS, dan JavaScript.

Ringan

Tanpa build tool, tanpa dependency tambahan. Tinggal jalan langsung di browser.

Routing Sederhana

Atur semua rute aplikasi cukup lewat satu file rute.json.

Modular

Header, footer, navbar, dan halaman dipisah rapi di folder masing-masing.
